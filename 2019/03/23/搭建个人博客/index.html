搭建环境

node.js

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境，可以在非浏览器环境下，解释运行 JS 代码
Node.js官网
下载安装包 v6.10.3 LTS,一直next,很快完成
也可以终端输入 sudo apt-get install nodejs
以及 sudo apt-get install npm
下载完成后,输入 node -v, npm -v 成功即可

git

ubuntu终端输入 sudo apt-get install git即可

hexo

终端输入 npm install hexo-cli -g
比较大,耐心等待
warning不用管

在终端进入Desktop或者你喜欢的文件目录下输入 git clone + 仓库地址(上图中复制的地址)
