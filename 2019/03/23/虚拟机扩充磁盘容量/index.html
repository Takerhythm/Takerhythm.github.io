虚拟机扩充磁盘容量

发表于 2019-03-23 | 分类于 虚拟机

关闭虚拟机

确保虚拟机处于电源关闭状态
ps:操作前先快照

点击编辑虚拟机设置

点击扩充


ps:如果报错:在部分链上无法执行所调用的函数，请打开父虚拟磁,请按下边方法操作
报错是因为磁盘系统为多文件系统,需要整合

报错处理方法

进入vmware文件目录下,按住shift+右键,点击在此处打开命令行窗口

输入如下命令
vmware-vdiskmanager.exe -r “D:\ubuntu1\xxx.vmdk” -t 0 “D:\ubuntu2\xxx.vmdk”
ps:文件夹路径要用双引号””,-r 选项后的路径为ubuntu所在文件夹下的磁盘路径,磁盘文件名如下图

ps:-t 0(零)选项后的路径为生成新的ubuntu2文件路径,ubuntu2文件需要手动创建否则报错
Convert: 100% done.
Virtual disk conversion successful.
终端显示100%后,将ubuntu1文件夹下的xxx.vmx文件移动到ubuntu2文件目录下(vmx后缀和vmdk不一样)
重新启动ubuntu即可,选择新的ubuntu2目录下的vmx文件打开
ps:重新点击扩充后,显示锁定文件失败的错误,是因为ubuntu运行时锁定系统文件,系统崩溃后没有解锁
解决方法如下:删除.lck文件目录下的所有文件即可

重启ubuntu

打开终端,输入sudo apt-get install gparted, 然后输入sudo gparted,界面如下

删除swap交换分区,然后删除extend扩展分区


新建扩展分区,然后新建交换分区(交换分区是扩展分区下的一部分),2G即可


重启交换分区

终端输入sudo swapon /dev/sda5
